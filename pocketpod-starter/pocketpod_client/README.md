# pocketpod_client

This is your Serverpod client. The code in here is mostly generated by
Serverpod, but you may want to make changes if you are adding modules to your
project.

PocketPod is built on [Serverpod](https://serverpod.dev). This client belongs to PocketPod `0.1.0`, uses Serverpod baseline `3.5.0-beta.10`, and is released as `v0.1.0+serverpod.3.5.0-beta.10`.

PocketPod was also initially inspired by [PocketBase](https://pocketbase.io), especially its lightweight SQLite deployment feel. PocketBase is not a dependency of this generated client.
