# pocketpod_flutter

A new Flutter project with Serverpod.

PocketPod is built on [Serverpod](https://serverpod.dev). This app belongs to PocketPod `0.1.0`, uses Serverpod baseline `3.5.0-beta.10`, and is released as `v0.1.0+serverpod.3.5.0-beta.10`.

PocketPod was also initially inspired by [PocketBase](https://pocketbase.io), especially its lightweight SQLite deployment feel. PocketBase is not a dependency of this Flutter app.

## Getting Started

This project is a starting point for a Flutter application that is using
Serverpod.

A great starting point for learning Serverpod is our documentation site at:
[https://docs.serverpod.dev](https://docs.serverpod.dev).

To run the project, first make sure that the server is running, then do:

```shell
flutter run
```
